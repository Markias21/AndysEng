# AndysEng

회화·글쓰기·표현 학습과 망각곡선 복습을 지원하는 영어 학습 PWA.
서버가 없는 정적 앱이라 GitHub Pages 등 정적 호스팅에 그대로 배포할 수 있고, 개인 컴퓨터에서도 돌릴 수 있다.

## 실행

```bash
npm run dev        # http://localhost:3000 (개발용 정적 서버, 의존성 없음)
npm test           # 전체 테스트 (node --test)
```

배포는 `public/` 폴더를 정적 호스팅(GitHub Pages 등)에 올리면 끝. 빌드 스텝이 없다.

## API 키

- 처음 열면 본인의 Anthropic API 키(`sk-ant-...`)와 잠금 비밀번호를 등록한다.
- 키는 비밀번호로 암호화(PBKDF2 → AES-GCM)되어 **이 브라우저의 localStorage에만** 저장된다. 평문 저장·서버 전송 없음.
- 앱을 열 때마다 비밀번호로 잠금을 해제하며, 상단 **🔒 잠금** 버튼으로 즉시 메모리에서 지울 수 있다.
- ⚠️ 공용 컴퓨터에서는 사용 후 "키 삭제 후 다시 설정"으로 키를 지울 것.
- API 사용량 절약: 표현은 호출 1회에 5개씩 받아 큐에 쌓고, 복습 퀴즈·통계·리포트는 AI를 전혀 쓰지 않는다.

## 기능

- **💬 회화** — AI가 먼저 말을 걸고, 문장마다 문법·자연스러움 피드백과 점수를 준다.
- **✍️ 글쓰기** — 논설 질문에 3~4문장 작성 → 첨삭 + 교정문 + 원어민 모범 답안 + 익혀둘 표현.
- **💡 표현** — 원어민 표현을 배우고 직접 예문을 만들어 첨삭받는다.
- **🔁 복습** — 표현·글쓰기에서 배운 표현이 덱에 자동으로 쌓이고, 망각곡선(1→3→7→14→30→60→120일)에 맞춰 빈칸 문제로 복습한다. 채점은 로컬이라 무료.
- **📊 통계** — 연속 학습일, 일별 주제 수·말한/쓴 문장 수, 평균 점수 성장 꺾은선그래프, 백업(내보내기/가져오기).

## 학습 기록 저장

- 모든 기록은 브라우저 localStorage에 남는다 (통계 탭에서 JSON 백업 가능).
- **🏁 공부 끝내기**를 누르면 지난 리포트 이후의 기록을 내 컴퓨터에 저장한다.
  - 크롬/엣지: 처음 한 번 폴더(예: 다운로드)를 고르면 그 안의 `AndysEng/` 폴더에 `YYYY-MM-DD.md`(리포트) + `.json`(원본 기록)으로 저장. 같은 날 2번째부터 `-2`, `-3` … 번호가 붙는다.
  - 사파리/파이어폭스: File System Access API 미지원이라 다운로드로 저장된다.
